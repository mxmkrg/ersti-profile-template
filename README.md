# Ersti‑Profil‑Template (HTML/CSS/JS)

Ein simples, editierbares Template für Erstis: Name, Hobbys, Unternehmen, Skills (1–10), Fun Fact. Keine Builds, nur statische Dateien.

## Nutzen
- Lokal öffnen: `index.html` doppelklicken
- Oder online in einem Editor (StackBlitz/CodePen/Replit)

## Download
- Auf GitHub: „Code“ → „Download ZIP“
- ZIP entpacken → `index.html` öffnen

## Online bearbeiten
- StackBlitz: „HTML (JavaScript)“ starten und die 3 Dateien anlegen/ersetzen.
- CodePen: New Pen → HTML/CSS/JS-Tabs befüllen.
- Replit: Create Repl → HTML/CSS/JS.

## Anpassen
- In `index.html` alle `<!-- EDIT ME: ... -->`-Stellen ändern.
- Skills: `data-score` (1–10) setzen; Anzeige passt sich an.

## GitHub Pages (optional)
- Repository → Settings → Pages
- Source: „Deploy from a branch“
- Branch: `main`, Folder: `/root`
- Die Seite ist dann unter `https://<user>.github.io/<repo>/` erreichbar.
